# YouTube Clone

A functional YouTube clone built with HTML, CSS, and JavaScript featuring multiple pages and video functionality.

## Features

- **Home Page**: Grid layout displaying video thumbnails with titles, channels, views, and upload dates
- **Video Player Page**: Full video player with video details, channel information, and suggested videos
- **Search Page**: Search functionality with filtered results
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **YouTube-like UI**: Dark theme with familiar YouTube styling

## Pages

1. **index.html** - Home page with video grid
2. **video.html** - Video player page with full video details
3. **search.html** - Search results page

## File Structure

```
youtube-clone/

├── images/             # Thumbnail images

```

## How to Use

1. **Open the Home Page**: Open `index.html` in your web browser
2. **Browse Videos**: Click on any video thumbnail to watch it
3. **Search Videos**: Use the search bar to find specific videos
4. **Navigate**: Use the sidebar to navigate between different sections
